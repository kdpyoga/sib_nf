Tentang Saya

Saya adalah Kartika Dwi Prayoga, mahasiswa Informatika di Universitas Negeri Surabaya (UNESA) dengan minat besar dalam pengembangan web dan sistem informasi. Saya memiliki pengalaman dalam membangun aplikasi berbasis web menggunakan framework CodeIgniter serta mengoptimalkan sistem database dengan MySQL.

Selain memiliki keterampilan teknis dalam pemrograman, saya juga tertarik pada desain antarmuka pengguna (UI/UX) untuk memastikan pengalaman pengguna yang optimal. Saya percaya bahwa teknologi yang baik harus mudah digunakan, efisien, dan memberikan solusi nyata bagi penggunanya.

Keahlian

  • HTML, CSS, & Bootstrap - Desain tampilan website yang responsif dan modern.
  • JavaScript & AJAX - Interaksi dinamis tanpa reload halaman.
  • Database MySQL - Perancangan dan optimasi database untuk efisiensi data.
  • UI/UX Design - Membuat desain antarmuka yang menarik dan user-friendly.
  • Scrum & Agile - Pengalaman bekerja dalam tim menggunakan metodologi Agile.

Pengalaman
